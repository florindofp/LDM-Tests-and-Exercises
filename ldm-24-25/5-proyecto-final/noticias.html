Acerca del proyecto

Este proyecto es una aplicación web que muestra el clima actual y noticias recientes utilizando APIs públicas. Forma parte del módulo de Lenguaje de Marcas en el ciclo DAW.

Objetivos

  • Practicar HTML5, CSS3 y JavaScript
  • Integrar APIs externas
  • Desarrollar un sitio web con múltiples páginas interconectadas

APIs utilizadas

  • WeatherAPI
  • GNews

Tecnologías empleadas

HTML, CSS, JavaScript, Bootstrap 5
